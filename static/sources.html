Data Sources

OCCRP Aleph
aleph.occrp.org
Global archive of investigative journalism data. Includes court records, corporate registries, leaked documents, and government gazettes from the Organized Crime and Corruption Reporting Project network.
Free, no API key required
OpenSanctions
opensanctions.org
Consolidated database of sanctioned entities and politically exposed persons (PEPs). Aggregates OFAC (US), EU consolidated list, UN Security Council, UK sanctions, and 30+ other watchlists into a single searchable API.
API key required (free tier available)
ICIJ OffshoreLeaks
offshoreleaks.icij.org
Database from the International Consortium of Investigative Journalists containing records from the Panama Papers, Paradise Papers, Pandora Papers, and other offshore leaks. Reveals offshore company structures, directors, and beneficial owners.
Free, no API key required
Pappers
pappers.fr
French business registry API. Searches company directors and beneficial owners across all French-registered companies. Returns SIREN numbers, legal forms, director roles, and beneficial ownership chains.
API key required (free tier with credit limits)
GDELT
gdeltproject.org
Global news monitoring system. Tracks news articles from thousands of sources worldwide in real-time. Searches for press mentions, media coverage, and public events related to a person or entity over the last 3 months.
Free, no API key required
